1) Посчитать факториал без рекурсии (через что на свое усмотрение)

2) Напишите функцию sum(n), которая для данного n вычисляет сумму чисел от 1 до n, например:
sum(1) = 1 
sum(2) = 2 + 1 = 3 
sum(3) = 3 + 2 + 1 = 6 

3) Создайте программу, которая фильтрует список строк и возвращает список, в котором есть только имя вашего друга.

Если имя имеет ровно 4 буквы, вы можете быть уверены, что он должен быть вашим другом!

Input = ["Ryan", "Kieran", "Jason", "Yous"]
Output = ["Ryan", "Yous"]

Примечание. Сохраните исходный порядок имен на выходе.

Дополнительно: Сделайте проверку на то, чтобы скрипт пропускал числа. В условие выше если число ввиде строки и содержит 4 символа то оно пройдет.

Input = ["Ryan", "Kieran", "Jason", "Yous", "1234"]
Output = ["Ryan", "Yous", "1234"] // ой, что-то не то :/
